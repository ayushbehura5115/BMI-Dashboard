# BMI Dashboard

## Overview
An interactive web-based BMI (Body Mass Index) Dashboard that allows users to calculate and visualize their BMI in real-time with a clean and responsive interface.

## Features
- Real-time BMI calculation based on height and weight input
- Dynamic color-coded BMI category indicators
- Fully responsive design for desktop and mobile devices
- Input validation and error handling for accurate results

## Tech Stack
- HTML5 — Structure and layout
- CSS3 — Styling and responsiveness
- JavaScript — Logic and interactivity

## How to Use
1. Open the HTML file in your browser
2. Enter your height and weight
3. View your BMI score and category instantly

## BMI Categories
BMI Below 18.5 — Underweight
BMI 18.5 to 24.9 — Normal
BMI 25 to 29.9 — Overweight
BMI 30 and above — Obese
